# 🚀 Brutalist Expense Tracker - Version 2.0

## New Features Added

### 1. 📊 Export/Import Data
- Export all transactions to CSV
- Export all transactions to JSON
- Import from CSV files
- Import from JSON files
- Merge or Replace modes
- Data validation

### 2. 📈 Analytics Dashboard
- Total transaction count
- Average income/expense
- Highest/lowest expenses
- Top 3 spending categories
- Color-coded stat cards

### 3. 🔍 Advanced Filtering
- Filter by category
- Filter by type (income/expense/all)
- Search descriptions and references
- Amount range filters (min/max)
- Real-time results
- Active filter indicators

### 4. 💰 Budget Tracking
- Set monthly budget limit
- Visual progress bar
- Warning at 80% usage
- Alert when budget exceeded
- localStorage persistence
- Color-coded status (green/yellow/red)

## Quick Start

```bash
git clone https://github.com/marktantongco/expense-tracker-brutalist.git
cd expense-tracker-brutalist
npm install
npm run dev
```

## Deploy

```bash
npm run deploy  # GitHub Pages
```
