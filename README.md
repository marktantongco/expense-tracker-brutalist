# 💰 EXPENSE TRACKER // BRUTALIST

A brutalist-style expense tracker built with React + Vite. Bold, aggressive design meets modern web tech.

![GitHub](https://img.shields.io/github/license/marktantongco/expense-tracker-brutalist)
![GitHub last commit](https://img.shields.io/github/last-commit/marktantongco/expense-tracker-brutalist)

## 🚀 Features

- **Bold Brutalist Design** - High contrast colors, chunky borders, and aggressive typography
- **Transaction Timeline** - Track income and expenses with visual clarity
- **Real-time Calculations** - Automatic totals and net balance
- **Responsive Design** - Works on mobile and desktop
- **CI/CD Ready** - GitHub Actions + Vercel integration

## 🛠️ Tech Stack

- React 18
- Vite
- Pure CSS (no frameworks)
- GitHub Actions (CI/CD)
- Vercel (Deployment)

## 📦 Quick Start

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

## 🌐 Deployment

### Quick Deploy to Vercel

1. **Option 1: Vercel Dashboard**
   - Go to https://vercel.com/new
   - Import `marktantongco/expense-tracker-brutalist`
   - Click Deploy

2. **Option 2: CLI**
   ```bash
   npm i -g vercel
   vercel --prod
   ```

3. **Option 3: Automated CI/CD**
   - See [SETUP_CICD.md](SETUP_CICD.md) for full setup
   - See [ADD_WORKFLOWS.md](ADD_WORKFLOWS.md) for GitHub Actions

## 🎨 Design Philosophy

This project embraces **Brutalist Web Design** principles:
- Raw, unpolished aesthetics
- High contrast color schemes (Red, Yellow, Green on Black/White)
- Bold typography and chunky borders
- No unnecessary animations or effects
- Function over form (but make it loud!)
